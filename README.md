# 🤖 Мульти-Агентная Система Генерации Проектов

Революционная система, которая использует **16 AI агентов** для автоматической генерации полноценных проектов на основе простого описания!

## ✨ Возможности

- 🎯 **16 специализированных AI агентов** - от проект-менеджера до тестировщика
- 🔄 **Циклическая разработка** - агенты работают итеративно для улучшения результата
- 🚀 **Полностью автоматизированное создание проекта** - от архитектуры до готового кода
- 📁 **Генерация файловой структуры** - создает реальные файлы проекта на вашем компьютере
- 🌐 **Поддержка разных технологий** - Python, JavaScript, React, Node.js, SQL и многое другое
- 📊 **Детальная отчетность** - отслеживание работы каждого агента
- 🎨 **Красивый интерфейс** - удобный терминальный интерфейс с Rich

## 🤖 Команда AI Агентов

### Управление и Планирование
- **Product Owner** - определяет требования продукта
- **Проект Менеджер** - анализирует требования и создает план
- **Системный Архитектор** - проектирует архитектуру системы

### Разработка
- **Backend Разработчик** - создает серверную логику
- **Frontend Разработчик** - создает пользовательский интерфейс  
- **Mobile Разработчик** - разрабатывает мобильное приложение
- **Database Engineer** - проектирует базы данных

### Качество и Безопасность
- **QA Тестировщик** - создает тесты
- **Security Специалист** - обеспечивает безопасность
- **Performance Engineer** - оптимизирует производительность
- **Code Reviewer** - проверяет качество кода

### Интеграция и Развертывание  
- **DevOps Engineer** - настраивает CI/CD и деплой
- **Integration Специалист** - интегрирует внешние сервисы

### Дизайн и Документация
- **UI/UX Дизайнер** - создает дизайн интерфейса
- **Technical Writer** - создает документацию
- **Data Scientist** - добавляет аналитику и ML

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
# Клонируйте проект или скачайте файлы
# Перейдите в директорию проекта

# Автоматическая установка (рекомендуется)
./run.sh

# Или вручную:
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Настройка API ключа

Получите API ключ от [DeepSeek](https://platform.deepseek.com/) и:

```bash
# Скопируйте файл примера
cp env_example.txt .env

# Отредактируйте .env и добавьте ваш API ключ
DEEPSEEK_API_KEY=your_actual_api_key_here
```

### 3. Запуск

```bash
# Интерактивный режим
python3 main.py

# Или используйте автоматический скрипт
./run.sh

# Быстрый режим (без интерфейса)
python3 main.py --project "Создать интернет магазин" --name "MyShop"
```

## 📊 Процесс работы

1. **Анализ требований** - Product Owner и Проект Менеджер анализируют ваше описание
2. **Проектирование** - Системный Архитектор создает техническую архитектуру
3. **Разработка** - Backend, Frontend и Mobile разработчики создают код
4. **Тестирование** - QA создает полный набор тестов
5. **Безопасность** - Security Специалист проверяет уязвимости
6. **Оптимизация** - Performance Engineer оптимизирует производительность
7. **Развертывание** - DevOps настраивает CI/CD
8. **Документация** - Technical Writer создает документацию
9. **Проверка качества** - Code Reviewer финальная проверка

## ⚙️ Конфигурация

В файле `config.py` вы можете настроить:

- Таймауты для агентов
- Роли и описания агентов
- Порядок выполнения агентов
- Настройки логирования

## 🔧 Расширенное использование

### Добавление новых агентов
```python
# В config.py добавьте нового агента
AGENT_ROLES["custom_agent"] = {
    "name": "Мой Агент",
    "description": "Описание агента",
    "prompt_template": "Промпт для агента: {project_description}"
}
```

### Использование через API
```python
from coordinator import AgentCoordinator

coordinator = AgentCoordinator()
project = await coordinator.start_project("Мой проект", "ProjectName")
success = await coordinator.execute_full_cycle()
```

## 🐛 Устранение неполадок

### Проблемы с API ключом
```bash
# Проверьте API ключ
python3 -c "from deepseek_client import deepseek_client; print(deepseek_client.health_check())"
```

### Проблемы с зависимостями  
```bash
# Переустановите зависимости
pip install --upgrade -r requirements.txt
```

### Логи
Проверьте логи в директории `logs/agents.log` для диагностики проблем.
