# 🚀 БЫСТРЫЙ СТАРТ

## За 3 минуты до первого проекта!

### 1. Установка (30 сек)
```bash
# Автоматическая установка
./run.sh

# Или вручную:
pip install -r requirements.txt
```

### 2. Настройка API ключа (2 мин)
1. Получите бесплатный API ключ на [DeepSeek](https://platform.deepseek.com/)
2. Скопируйте `env_example.txt` в `.env`
3. Вставьте ваш API ключ в `.env`

```bash
cp env_example.txt .env
# Отредактируйте .env файл
```

### 3. Запуск (30 сек)
```bash
# Интерактивный режим
python3 main.py

# Быстрый режим
python3 main.py --project "Создать интернет магазин" --name "MyShop"

# Демонстрация (без API ключа)
python3 demo.py
```

```bash

```

## 🔧 Если что-то не работает

1. **Проблемы с Python:** Убедитесь что установлен Python 3.8+
2. **Проблемы с API:** Проверьте API ключ в `.env`
3. **Проблемы с зависимостями:** Запустите `pip install --upgrade -r requirements.txt`
